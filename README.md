# Issue Tracker
A system to track bugs and assign to other users.

## Tech Stack
 - Next.js 15 (App Router)
 - TypeScript
 - Supabase

## Installation Steps and Modules

 1. To create a new Next.js app, run: `npx create-next-app@latest`
 2. Install React Icons: `npm i react-icons` or `npm install react-icons`
